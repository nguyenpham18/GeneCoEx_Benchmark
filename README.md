## Project Overview

This project benchmarks different pipelines for constructing gene co-expression networks (GCNs) from bulk RNA-seq data. We systematically compare:

- **Similarity metrics:** Pearson correlation, Spearman correlation, Proportionality (ρ)
- **Adjacency construction:** Soft thresholding (weighted) and Hard thresholding (unweighted)

All analyses are performed on the GTEx v11 Brain Cortex RNA-seq dataset.

## Data Download

Data files are not included in this repository due to file size. Download them manually and place them in the correct folders.

### 1. GTEx v11 Brain Cortex Raw Counts
- Go to: https://gtexportal.org/home/downloads/adult-gtex/bulk_tissue_expression
- Look at GTEx Analysis V11 section
- Select: **Gene read counts by tissues**
- Download: `gene_reads_v11_brain_cortex.gct.gz`
- Place in: `data/raw/gene_reads_v11_brain_cortex.gct.gz`

### 2. GENCODE v47 Gene Annotation
- Go to: https://gtexportal.org/home/downloads/adult-gtex/reference
- Select: **GTEx Analysis V11 → Reference Tables**
- Download: `gencode.v47.genes.gtf` (the second file)
- Place in: `data/annotation/gencode.v47.genes.gtf`

## How to Run

Run the scripts in order from the project root directory.

### Step 1 — Preprocessing
```bash
python src/preprocess.py
```
**Input:** `data/raw/gene_reads_v11_brain_cortex.gct.gz`  
**Output:** `data/processed/preprocessed_counts.csv`  
**What it does:**
- Loads raw GTEx counts
- Removes low-quality samples (≥50% zero expression)
- Filters lowly expressed genes (CPM ≥ 1 in ≥20% samples)
- Filters to protein-coding, lncRNA, antisense genes only
- Applies CTF normalization + log1p transformation

Expected output:
```
Final matrix: 17578 genes × 125 samples
```

---

### Step 2 — Pearson Similarity Matrix
```bash
python src/pearson.py 
  --input data/processed/preprocessed_counts.csv \
  --output data/similarity/pearson_similarity.csv
```
**Input:** `data/processed/preprocessed_counts.csv`  
**Output:** `data/similarity/pearson_similarity.csv`  
**Note:** This may take several minutes. Output is a 17,578 × 17,578 matrix (~2 GB).
